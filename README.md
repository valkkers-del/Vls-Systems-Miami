# 🛡 VLS Systems® - Miami Health Hub

Aplicação profissional de gestão integrada de pacientes e rotas de transporte, desenvolvida com arquitetura **Offline First** e design **Mobile First**.

## Visão Geral

VLS Systems® Miami Health Hub é uma solução completa para gerenciamento de operações clínicas e logísticas em Miami. Desenvolvida com foco em performance, segurança de dados e experiência do usuário, a aplicação oferece funcionalidade contínua mesmo em ambientes sem conectividade.

## ✨ Funcionalidades Principais

- **Gestão de Clínica:** Cadastro completo e acompanhamento de pacientes com persistência local segura
- **Modo Motorista:** Gerenciamento intuitivo de rotas e logística operacional
- **Offline First:** Aplicação funciona completamente offline com sincronização automática
- **Persistência de Dados:** LocalStorage para manutenção segura de informações
- **Design Responsivo:** Interface otimizada para dispositivos móveis e tablets
- **Interface Moderna:** Sistema de design profissional com componentes interativos

## 🛠 Tecnologias Utilizadas

- **HTML5** — Estrutura semântica e moderna
- **CSS3** — Design system responsivo com custom properties
- **JavaScript (ES6+)** — Lógica de negócio e persistência
- **LocalStorage API** — Armazenamento cliente-side seguro

## 📱 Acesso à Aplicação

A aplicação está disponível através do **GitHub Pages**:

🔗 **[VLS Systems® Miami Health Hub](https://valkkers-del.github.io/Vls-Systems-Miami/)**

## 🚀 Como Usar

1. Acesse a URL pública do projeto
2. Escolha o modo de operação: **Clínica** ou **Motorista**
3. No modo **Clínica**, adicione novos pacientes usando o botão "+ Novo Paciente"
4. Todos os dados são salvos automaticamente no dispositivo
5. A aplicação funciona completamente offline

## 📋 Requisitos

- Navegador moderno com suporte a:
  - HTML5
  - CSS3 (Custom Properties)
  - JavaScript ES6+
  - LocalStorage API
