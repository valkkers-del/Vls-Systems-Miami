# VLS Systems - Miami Health Hub

Projeto desenvolvido para a gestão integrada de pacientes e rotas de transporte, focado em alta performance e experiência do usuário (UX).

## Funcionalidades
- **Gestão de Clínica:** Cadastro e acompanhamento de pacientes com persistência local de dados.
- **Modo Motorista:** Gerenciamento de rotas e logística para operações em Miami.
- **Offline First:** Aplicação desenvolvida com foco em funcionalidade contínua, utilizando LocalStorage para salvar dados localmente.

## Tecnologias Utilizadas
- HTML5, CSS3, JavaScript (ES6+).
- Design responsivo com foco em dispositivos móveis.
